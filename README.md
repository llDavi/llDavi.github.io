# davi — ghost in the wire

Sito del podcast **Ghost in the Wire** di davi.

## stack

- [VitePress](https://vitepress.dev/) — static site generator
- Vue 3 SFC — landing page custom (`Home.vue`)
- JetBrains Mono + Inter

## sviluppo locale

```bash
npm install
npm run dev
```

## build

```bash
npm run build
npm run preview
```
