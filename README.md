# Davi - Cybersecurity Research Portfolio

A modern, responsive cybersecurity research portfolio built with cutting-edge web technologies.

## 🚀 Tech Stack

- **HTML5** - Semantic markup and accessibility
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **Swiper.js** - Touch-enabled carousels and sliders
- **AOS (Animate On Scroll)** - Smooth scroll animations
- **Inter Font** - Modern, highly readable typography
- **JetBrains Mono** - Monospace font for technical content

## 🎨 Design Features

- **Dark SaaS Theme** - Professional dark color palette
- **Responsive Design** - Mobile-first approach with breakpoints
- **Smooth Animations** - AOS-powered scroll animations
- **Interactive Carousels** - Netflix-style article carousels
- **Hover Effects** - Subtle animations and transitions
- **Multilingual Support** - English, Italian, and Spanish

## 📱 Pages

- **Home** (`index.html`) - Hero section, research carousel, statistics
- **Tools** (`tools.html`) - Cybersecurity tools organized by category
- **About** (`about.html`) - Philosophy, expertise areas, community involvement

## 🛠️ Local Development

```bash
# Start local server
python3 -m http.server 8000

# Open in browser
http://localhost:8000/index.html
```
